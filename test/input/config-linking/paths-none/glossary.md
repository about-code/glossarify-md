# Glossary

## Sibling

...MUST be referred to in './document.md' with `#sibling`
AND *is expected* to be no longer navigable in a multi-file output fileset.

## Second-Level-Parent

...MUST be referred to in './sub-1/sub-2/document.md' with `#second-level-parent`
AND *is expected* to be no longer navigable in a multi-file output fileset.
