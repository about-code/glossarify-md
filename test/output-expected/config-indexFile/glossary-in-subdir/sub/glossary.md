## [Term](#term)

GIVEN a term "[Term][1]"
  AND option 'indexFile' is './index.md'
  AND the glossary file is in './sub/glossary.md'
  AND config option 'linking' is 'relative'
THEN the term MUST be linked with a path './sub/glossary.md#term'.
